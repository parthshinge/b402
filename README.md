## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/parthshinge/b402.git
   cd b402
   ```

2. **Open in browser**
   - Simply open `index.html` in your preferred web browser
   - Or use a local server like `npx http-server -p 8000` and visit `http://localhost:8000`
